# 2048-java

A simple version of the 2048 game in Java, complete with a sophisticated UI.

## Current Status

Currently, the project is undergoing development. The program only runs in the console for now, but a UI will be created shortly.

## How to Run

To play the game, run the following commands:
```bash
# javac is used to compile the java code, and java is used to run the resulting class file
git clone https://github.com/DEV-ajthedj/2048-java.git
cd 2048-java
javac *.java
java Main
```
Make sure to install the latest JDK and git to run these commands!
